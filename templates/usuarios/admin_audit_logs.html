Para habilitar el registro de auditoría en tiempo real, debes crear la tabla en tu base de datos utilizando phpMyAdmin en XAMPP.

Copia el siguiente script SQL, abre phpMyAdmin, selecciona la base de datos de tu proyecto, ve a la pestaña SQL, pega el código y presiona Continuar / Ejecutar. 

CREATE TABLE IF NOT EXISTS `admin_audit_log` (
  `id_log` INT AUTO_INCREMENT PRIMARY KEY,
  `id_admin` INT NULL,
  `accion` VARCHAR(100) NOT NULL,
  `objetivo` VARCHAR(255) NULL,
  `detalles` TEXT NULL,
  `ip_address` VARCHAR(45) NULL,
  `fecha` DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (`id_admin`) REFERENCES `tblusuarios`(`id_users`) ON DELETE SET NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
